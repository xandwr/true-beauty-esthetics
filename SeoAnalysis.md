## ⚠️ Warnings:

- **Site title is too long**  
  *Current:*  
  `Waxing, Facials, Eyelash Extensions & More | True Beauty Esthetics` (66 characters)  
  *Best practice:* 20–60 characters  
  > ✅ 100% of the top 100 websites follow this range

- **Meta description is too short**  
  *Current:*  
  `Waxing, Facials, Eyelash Exts., Lash Lift & Tints, Brow Lamination & Tints, Manicures, Pedicures! Located in NW Edmonton, AB (YEG)` (130 characters)  
  *Best practice:* 150–220 characters  
  > ✅ 92% of the top 100 websites meet this range

- **Keyword usage test failed**  
  > Primary keywords should appear in the title tag, meta description, and heading tags.  
  *Detected keywords:* `beauty`, `true`, `esthetics`, `book`, `cart`

- **No H1 headings detected**  
  > H1s help indicate the primary topics of a page for search engines and assist with indexing.

- **No analytics scripts detected**  
  > Google Analytics is free and essential for monitoring SEO performance.

- **No social media integration detected**  
  > Social signals (like open graph tags) are increasingly used to validate authority.

- **HTML size is too large**  
  *Current:* 53.33 KB  
  *Recommended:* <33 KB  
  > Slower loads can hurt conversion and retention.

- **DOM size exceeds recommendations**  
  *Current:* 2,133 nodes  
  *Recommended:* <1,500  
  > High DOM size affects performance and page speed.

- **Too many HTTP requests**  
  > More than 20 can significantly slow down load times.

  | Content Type | % of Page | Size   |
  |--------------|-----------|--------|
  | Images       | 60.8%     | 3.02MB |
  | JavaScript   | 34.6%     | 1.72MB |

- **Images not served in modern formats**  
  > Use WebP, JPEG 2000, or JPEG XR instead of PNG/JPEG for better compression.

- **Render-blocking resources present**  
  > These prevent the page from loading until all files finish.

- **Canonical URL mismatch**  
  > `https://truebeautyesthetics-yeg.square.site/` and  
    `https://www.truebeautyesthetics-yeg.square.site/`  
    should resolve to the same address.

- **Strict-Transport-Security header missing**  
  > Helps enforce HTTPS for improved security.

- **Structured data not detected (JSON-LD or Microdata)**  
  > Structured data improves search result visibility (e.g. review stars, FAQs).

- **SPF record missing**  
  > Without it, spammers can spoof your domain via email.
